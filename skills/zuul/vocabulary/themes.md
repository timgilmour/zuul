# Vocabulary — Themes

Per-genre tone, palette, and descriptor language now lives in the structured data: `genre.json` (7 genres) and `subgenres.json` (35 sub-genres — each with `tone`, `visual` palette deltas, `descriptors[]`, and `prompt_fragments[]`). Look themes up there by genre / sub-genre `id`.

This file keeps only the **cross-cutting mood/atmosphere language** below — it applies across every genre.

## Mood and atmosphere language

Use these phrases to anchor emotional tone in prompts. More precise than adjectives like "dark" or "cool."

| Mood | Prompt phrases |
|------|----------------|
| **Mysterious** | "mysterious and eerie," "atmospheric tension," "shadowy intrigue" |
| **Foreboding** | "ominous weight," "dread atmosphere," "wrong stillness" |
| **Serene** | "peaceful and still," "calm diffuse light," "meditative quiet" |
| **Heroic** | "luminous, mythic scale," "noble bearing," "righteous energy" |
| **Brutal** | "raw power," "unrefined force," "scarred pragmatism" |
| **Desolate** | "abandoned and weathered," "dust and silence," "forgotten" |
| **Corrupted** | "wrongness beneath the surface," "something that was once holy," "decay at the edges" |
| **Ancient** | "worn by centuries," "patina of age," "pre-historical weight" |
