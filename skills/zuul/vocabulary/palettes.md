# Vocabulary — Color Palettes

Named palette language for prompt use. Never use hex codes in prompts — use these phrases instead.

> **Rule:** `#1A8A9B` renders as visible text in the image. "Deep teal" works.

## Palette types

| Palette | Prompt phrases |
|---------|----------------|
| **Warm** | "warm tones," "golden hour colors," "amber and burnt sienna," "sun-weathered" |
| **Cool** | "cool blues and slate greys," "icy tones," "winter light," "overcast diffuse" |
| **Muted / Desaturated** | "muted earth tones," "desaturated palette," "washed-out," "faded" |
| **Vibrant / Saturated** | "deeply saturated colors," "bold high-energy palette," "vivid, punchy" |
| **Monochromatic** | "monochromatic scheme," "tonal range in [color]," "single-hue variations" |
| **High contrast** | "high-contrast," "deep shadows, bright highlights," "stark light" |
| **Soft / Diffuse** | "soft diffuse light," "gentle tonal transitions," "low contrast, even exposure" |

## Fantasy palettes

| Style | Palette language |
|-------|-----------------|
| Dark fantasy | "tarnished iron, dried blood, deep shadow, muted jewel tones, soot-black" |
| High fantasy | "bright steel, gold leaf, sapphire blue, warm oak, luminous white" |
| Celtic / Iron Age | "woad blue, bronze, undyed linen, rust iron, deep forest green" |
| Necromantic | "necrotic yellow-green, bone white, deep violet, abyssal black" |
| Fey / Arcane | "iridescent, shifting hues, bioluminescent accents, soft violet-gold" |

## Sci-fi palettes

| Style | Palette language |
|-------|-----------------|
| Gritty / Industrial | "gunmetal, scuffed matte black, warning orange, hydraulic yellow, soot" |
| Clean / Corporate | "matte white, brushed aluminium, cool cyan accent, sterile grey" |
| Biopunk | "organic greens, wet membrane sheen (use sparingly), dark crimson" |
| Retro-futurist | "chrome accents, turquoise, coral, warm beige, rounded forms" |

## Horror palettes

| Style | Palette language |
|-------|-----------------|
| Cosmic | "deep violet, bioluminescent green, abyssal black, wrong proportions" |
| Gothic | "ash grey, deep crimson, tarnished silver, candlelight amber" |
| Body horror | "necrotic green, exposed tissue red, pallid skin, sickly yellow" |
