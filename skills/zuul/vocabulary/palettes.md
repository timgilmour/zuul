# Vocabulary — Color Palettes

Named palette language for prompt use. Never use hex codes in prompts — use these phrases instead.

> **Rule:** `#1A8A9B` renders as visible text in the image. "Deep teal" works.

## General palette language

| Palette | Prompt phrases |
|---------|----------------|
| **Warm** | "warm tones," "golden hour colors," "amber and burnt sienna," "sun-weathered" |
| **Cool** | "cool blues and slate greys," "icy tones," "winter light," "overcast diffuse" |
| **Muted / Desaturated** | "muted earth tones," "desaturated palette," "washed-out," "faded" |
| **Vibrant / Saturated** | "deeply saturated colors," "bold high-energy palette," "vivid, punchy" |
| **Monochromatic** | "monochromatic scheme," "tonal range in [color]," "single-hue variations" |
| **High contrast** | "high-contrast," "deep shadows, bright highlights," "stark light" |
| **Soft / Diffuse** | "soft diffuse light," "gentle tonal transitions," "low contrast, even exposure" |
